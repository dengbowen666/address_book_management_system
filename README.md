## 项目概述：简要介绍项目的目的、功能和特点，以及解决的问题或提供的价值。
### 具体需求
1. menu设计
2. 给通讯录addressBook 装载数据
3. 按照姓名对通讯录排序
4. 选择某种方式（姓名，电话，小标签……）查找人，并输出符合要求的用户信息人的地址、电话号码、出生日期等信息。
5. 增、删、修改联系人信息。
